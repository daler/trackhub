class trackhub.constants.Parameter(param, desc, values)[source]¶

Bases: object

Represents a generic parameter. Validates based on the provided possible values, e.g.:

>>> Parameter("name", "the name to use", str).validate("asdf")
True

>>> Parameter("name", "the name to use", int).validate(999)
True

>>> Parameter("name", "the name to use", str).validate(999)
False
